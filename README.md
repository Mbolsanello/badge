## Comandos Git
- git clone (clonar um repositório)
- git branch (mostrar branch atual)
- git checkout -b (criar uma nova branch)
- git status (ver mudanças que tiveram no arquivo)
- git commit -m (salvar as alterações)
- git push origin (enviar alterações ao repositório remoto)
- git add . (adiciona um único arquivo no stage changes)
